Script SQL Server - Sistema de Inventario

En este ejercicio se realizó el proceso de normalización y posterior implementación de una base de datos en Microsoft SQL Server a partir de una ficha de inventario tecnológico.

Ficha de Inventario Utilizada

La siguiente ficha fue utilizada como documento base para identificar entidades, atributos, relaciones y posteriormente construir el modelo relacional normalizado en SQL Server.

Introducción

Luego del proceso de análisis y normalización del formulario, se desarrolló la estructura completa de la base de datos utilizando sentencias SQL en Microsoft SQL Server.

El desarrollo incluye la creación de la base de datos, construcción de tablas, relaciones entre entidades e inserción masiva de registros de prueba para validar el funcionamiento del sistema.

1. Script de la Base de Datos

Sentencia principal utilizada para crear la base de datos en Microsoft SQL Server. 

CREATE DATABASE InventarioComputadores;

📌 Script de la tabla Usuario 

USE InventarioComputadores;
GO

CREATE TABLE USUARIO (
    id_usuario VARCHAR(20) NOT NULL,
    id_dependencia VARCHAR(20) NOT NULL,
    nombre_usuario VARCHAR(100) NOT NULL,
    
    -- Definimos la llave primaria compuesta (ambas columnas)
    CONSTRAINT PK_USUARIO PRIMARY KEY (id_usuario, id_dependencia),
    
    -- Definimos la llave foránea que la conecta con DEPENDENCIA
    CONSTRAINT FK_USUARIO_DEPENDENCIA FOREIGN KEY (id_dependencia) REFERENCES DEPENDENCIA(id_dependencia)
);
GO

GO

CREATE TABLE EQUIPO (
    id_equipo VARCHAR(20) NOT NULL,
    id_usuario VARCHAR(20) NOT NULL,
    id_dependencia VARCHAR(20) NOT NULL,
    tipo VARCHAR(50),
    marca VARCHAR(50),
    modelo VARCHAR(50),
    serie VARCHAR(100),

    CONSTRAINT PK_EQUIPO PRIMARY KEY (id_equipo, id_usuario, id_dependencia),
