Semana 8 – Unidad II

Ver resumen

Temas tratados

En esta semana se desarrolló la introducción al uso de consultas en SQL, aprendiendo cómo obtener información almacenada dentro de una base de datos mediante la instrucción SELECT, utilizada para leer y mostrar datos contenidos dentro de una o varias tablas.

  • Preparación del escenario: organización previa del entorno antes de comenzar a realizar consultas SQL.
  • Consultas a las tablas: uso de la sentencia SELECT para recuperar información almacenada.
  • Estableciendo filtros de fila: aplicación de condiciones para mostrar únicamente determinados registros.
  • Manipulación de valores NULL: tratamiento de campos vacíos o sin información dentro de una consulta.
  • Funciones para manipulación de fechas: funciones utilizadas para consultar, convertir y mostrar fechas.
