📘 Semana 16

Conectividad, Programación y Referencia Técnica de Transact-SQL en SQL Azure

En esta última semana se estudiaron los mecanismos de conectividad entre aplicaciones y SQL Azure, así como la programación mediante Transact-SQL y las principales sentencias utilizadas para la definición, manipulación y administración de bases de datos en la nube.

Además, se revisaron aspectos relacionados con la compatibilidad entre SQL Server y SQL Azure, los tipos de datos soportados, procedimientos almacenados, control de errores y la administración de permisos mediante el lenguaje DCL.

Conectividad y Programación

Conexión de aplicaciones .NET, ASP.NET y Entity Framework con SQL Azure.
Integración mediante servicios web (WCF), PHP y Java (JDBC).
Tipos de datos, operadores y funciones disponibles en Transact-SQL.
Compatibilidad de instrucciones y procedimientos almacenados en SQL Azure.

Definición, Manipulación y Seguridad

Lenguaje de Definición de Datos (DDL): CREATE, ALTER y DROP.
Lenguaje de Manipulación de Datos (DML) y control de transacciones.
Administración de permisos mediante GRANT, REVOKE y DENY.
Buenas prácticas para el desarrollo y administración de bases de datos en SQL Azure.
