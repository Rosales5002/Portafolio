📘 Semana 14

Automatización y Administración Avanzada en SQL Server

Durante esta semana se estudiaron las principales herramientas para automatizar tareas administrativas en SQL Server mediante SQL Server Agent, así como la implementación de planes de mantenimiento para garantizar el correcto funcionamiento de las bases de datos.

Además, se abordó la automatización utilizando T-SQL y PowerShell, el monitoreo proactivo mediante alertas y registros del sistema, y una introducción al uso de Amazon Web Services (AWS) como plataforma de servicios en la nube.
