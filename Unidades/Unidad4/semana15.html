📘 Semana 15

Arquitectura, Administración y Fundamentos de la Base de Datos en la Nube

Durante esta semana se estudiaron los fundamentos de SQL Azure, comprendiendo su arquitectura, administración y funcionamiento dentro de la plataforma Microsoft Azure.

Asimismo, se revisaron conceptos relacionados con la gestión de bases de datos, seguridad, monitoreo y desarrollo de aplicaciones utilizando los primeros capítulos del libro de SQL Azure.

Arquitectura y Administración

Arquitectura general y funcionamiento de SQL Azure.
Modelo de almacenamiento y aprovisionamiento de bases de datos.
Administración de bases de datos, logins y usuarios.
Migración inicial de bases de datos hacia SQL Azure.

Supervisión y Desarrollo

Copia y administración de bases de datos entre servidores.
Monitoreo mediante Dynamic Management Views (DMVs).
Configuración del Firewall y seguridad del servidor.
Desarrollo de aplicaciones y compatibilidad con herramientas Microsoft.
