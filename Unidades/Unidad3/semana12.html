📘 Semana 12

Respaldo y Recuperación de Bases de Datos

Durante esta semana se estudiaron las principales estrategias para proteger la información almacenada en una base de datos mediante copias de seguridad y procedimientos de recuperación. Se revisaron los diferentes tipos de respaldos disponibles en SQL Server y su importancia para garantizar la continuidad del servicio ante fallos o pérdidas de información.

Asimismo, se desarrollaron procedimientos de restauración utilizando las instrucciones BACKUP DATABASE y RESTORE DATABASE, además de conocer los planes de mantenimiento y las políticas de retención utilizadas en ambientes empresariales.

Temas Desarrollados

Estrategias de respaldo completo, diferencial y transaccional.
Restauración de bases de datos en distintos escenarios.
Verificación y mantenimiento de copias de seguridad.
Uso de BACKUP DATABASE y RESTORE DATABASE.

Administración de Recuperación

Planes automáticos de mantenimiento.
Políticas de retención de respaldos.
Recuperación de información hasta un punto específico en el tiempo.
Buenas prácticas para garantizar la disponibilidad de los datos.
