📘 Semana 11

Seguridad y Control de Acceso en SQL Server

Durante esta semana se abordaron los principales mecanismos de seguridad implementados en Microsoft SQL Server para proteger la información almacenada en las bases de datos. Se estudiaron los diferentes métodos de autenticación, administración de usuarios, control de permisos y técnicas de cifrado que permiten garantizar la confidencialidad e integridad de los datos.

Además, se revisaron herramientas de auditoría y monitoreo que permiten registrar eventos y fortalecer la seguridad dentro del entorno del servidor.

Temas Desarrollados

Autenticación mediante SQL Server y Windows.
Configuración de cuentas de servicio del servidor.
Creación y administración de roles.
Administración de permisos con GRANT, DENY y REVOKE.

Seguridad de la Información

Protección de datos mediante TDE y Always Encrypted.
Auditoría utilizando SQL Server Audit.
Monitoreo de eventos y actividades del servidor.
Buenas prácticas para la administración segura de bases de datos.
