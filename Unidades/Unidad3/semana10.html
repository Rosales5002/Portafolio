📘 Semana 10

Administración Esencial en SQL Server

Durante esta semana se estudiaron los principales aspectos relacionados con la administración de bases de datos en Microsoft SQL Server, comprendiendo la estructura de almacenamiento, configuración de bases de datos, seguridad, recuperación de información y herramientas de monitoreo.

Asimismo, se introdujo el uso de Microsoft Azure como plataforma en la nube para la administración de bases de datos y la creación de tablas mediante SQL Server, complementando los conocimientos adquiridos durante el desarrollo del curso.

Administración de SQL Server

Estructura de almacenamiento y archivos de datos.
Propiedades y configuraciones de bases de datos.
Modelos de recuperación (Simple, Full y Bulk-Logged).
Administración de usuarios y roles de seguridad.
Asignación de permisos y políticas de acceso.

Herramientas y Administración

Monitoreo mediante SQL Server Activity Monitor.
Introducción al uso de SQL Server Agent.
Automatización de tareas administrativas.
Introducción a Microsoft Azure.
Creación y administración de tablas mediante SQL Server.
