Cuestionario Resuelto

Preguntas desarrolladas sobre arquitecturas de bases de datos, modelo relacional y conceptos fundamentales.

1. ¿Quién propuso por primera vez el modelo relacional?
a) Codd
Edgar F. Codd formuló el modelo relacional en 1970, estableciendo las bases teóricas de las bases de datos modernas.
2. ¿Cuál es el sistema de gestión de base de datos basado en microcomputadora más ampliamente utilizado?
b) Access de Microsoft
Microsoft Access fue diseñado para computadoras personales y destacó por su facilidad de uso.
3. ¿Qué representa físicamente el modelo relacional?
a) Una tabla
Los datos en el modelo relacional se organizan en tablas compuestas por filas y columnas.
4. ¿Qué es una tupla en el contexto de una relación?
b) Una fila
Cada tupla representa un registro completo dentro de una tabla.
5. ¿Cuál es la característica de cada celda en una tabla que representa una relación?
b) Contiene solo un valor
Cada campo debe contener un único valor atómico.
6. ¿Quién propuso por primera vez el modelo relacional?
a) Codd
Edgar F. Codd es reconocido como el creador del modelo relacional.
7. ¿Cuál es el sistema de gestión de base de datos basado en microcomputadora más ampliamente utilizado?
b) Access de Microsoft
Fue muy popular en microcomputadoras por su accesibilidad.
8. ¿Qué representa físicamente el modelo relacional?
a) Una tabla
El modelo relacional estructura la información en tablas.
9. ¿Qué es una tupla en el contexto de una relación?
b) Una fila
Una tupla equivale a una fila o registro.
10. ¿Cuál es la característica de las relaciones en el modelo relacional?
c) Cada tupla es distinta
No deben existir filas duplicadas para mantener la integridad.
11. ¿Qué es el grado de una relación?
b) El número de atributos
El grado indica cuántas columnas tiene una tabla.
12. ¿Qué se entiende por cardinalidad en una relación?
b) El número de tuplas
La cardinalidad corresponde a la cantidad de filas o registros.
13. ¿Qué se considera un atributo en el modelo relacional?
b) Un nombre de columna
Los atributos representan propiedades de los datos.
14. ¿Qué se dice de los valores de los atributos en una relación?
b) Son atómicos
Cada valor debe ser indivisible dentro del campo.
15. ¿Qué se utiliza para almacenar información sobre los objetos en la base de datos?
d) Tablas
Las tablas son la estructura principal para almacenar información.
